# ML_2026
Технологии машинного обучения 2026

В состав проекта входят работы по следующим темам:

EDA - разведовательный анализ данных
EDA Missing - базовые и продвинутые методы кодирования категориальных признаков и обработки пропущенных значений
Base Classification - решение задачи бинарной классификации

Используемые технологии:

SeaBorn
Pandas
Matplotlib
